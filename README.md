Claude Code AI Security Project:- 
A hands-on project implementing a 3-layer security system on Claude Code to prevent data deletion and unauthorised access to secrets.

What This Project Does?
This project puts three security controls in place, on top of Claude Code so that Claude cannot delete critical data, read sensitive files like passwords and API keys, or behave in ways that could harm your system. The three layers work together so that if one is bypassed, the others still hold.
